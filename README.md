# XMAX Logbook

สมุดบันทึกการเติมน้ำมัน & การบำรุงรักษา สำหรับ **Yamaha XMAX 300 (CZD300-A / B5X)**

แอป **offline-first, single-file, ไม่มี backend** — ข้อมูลทั้งหมดเก็บในเครื่อง (localStorage)
ออกแบบสำหรับ iPhone (safe-area, dark mode, ติดตั้งเป็น PWA ลงโฮมสกรีนได้)

## ฟีเจอร์
- 🏠 **หน้าหลัก** — เลขไมล์, การ์ดแจ้งเตือนซ่อมบำรุงตามความเร่งด่วน, สถิติด่วน
- ⛽ **น้ำมัน** — บันทึกการเติม, คำนวณ กม./ลิตร แบบเต็มถัง→เต็มถัง อัตโนมัติ
- 🔧 **ซ่อมบำรุง** — ตารางบำรุงตามคู่มือ (15 รายการ + พ.ร.บ./ภาษี/ประกัน), ประวัติแบบ timeline
- 📊 **สรุป** — กราฟค่าใช้จ่ายรายเดือน + อัตราสิ้นเปลือง (SVG เขียนเอง ไม่พึ่ง lib)
- ⚙️ **ข้อมูลรถ** — ข้อมูลอ้างอิงจากคู่มือ, ตั้งค่า, Export/Import JSON

## การใช้งาน
เปิดไฟล์ [`index.html`](index.html) ในเบราว์เซอร์ได้เลย — เป็นไฟล์เดียวจบ ฝัง React + Tailwind + กราฟทั้งหมด ไม่ต้องต่อเน็ต

### ติดตั้งลง iPhone
เปิดลิงก์ใน Safari → ปุ่มแชร์ → **"เพิ่มไปยังหน้าจอโฮม"**

## การพัฒนา
แก้โค้ดที่ [`src/index.src.html`](src/index.src.html) (เวอร์ชันอ่านง่าย ใช้ Babel runtime) แล้ว build:

```bash
node build.mjs
```

`build.mjs` จะ pre-compile JSX + ฝัง React/Tailwind ลงเป็น `index.html` ตัวเดียวที่เปิดออฟไลน์ได้ 100%
(ดาวน์โหลด vendor libs อัตโนมัติครั้งแรก)

## เทคโนโลยี
React 18 · Tailwind CSS · กราฟ SVG เขียนเอง · localStorage · PWA (manifest + service worker)
